# merry  

极简hexo主题，真的啥也没有ww  
支持latex  

## Demo  

[Demo](https://anillc.cn)

## Usage  

```bash
cd hexo
git clone https://github.com/Anillc/merry.git themes/merry
```

修改\_config.yml  

```yml
theme: merry
```
