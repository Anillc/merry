## 这个是什么仓库？

这是一个非官方的merry主题仓库，官方主题请见[这里](https://github.com/anillc/merry)，遵循MIT

这个fork仓库主要是更新我所修改后的merry主题

即便已经违背了「真的什么都没有啦」

版本号按照官方仓库标注的版本

## 关于「极简hexo主题，真的啥也没有ww」这句话的理解

在我看来吧，极简，除了没有花里胡哨的特效（例如鼠标点击，live2d等）外

也没有一些不必要的功能吧

后续修改的主题也会优先考虑“这个功能需不需要，使用频率高不高”这样
